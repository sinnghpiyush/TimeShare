TimeShare – Peer-to-Peer Career Guidance Platform

Project Overview

TimeShare is a peer-to-peer career guidance platform designed to connect students directly with mentors, seniors, and professionals.
The platform allows students to receive real-world career advice through chat, video calls, and mentorship sessions.

The goal of the project is to remove the confusion caused by advertisements, agents, and misleading online information by providing direct communication with experienced individuals.

---

Features

- Student and Mentor Registration
- Secure Login System
- Mentor Profile Creation
- Mentor Discovery and Filtering
- Session Booking System
- Blog / Article Section
- Private Real-time Chat System
- Dashboard for Students and Mentors
